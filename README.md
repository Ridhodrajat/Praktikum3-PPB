## Additional Information

Repository ini berisi seluruh dokumentasi kode hasil pengerjaan Praktikum 3 pada mata kuliah Pemrograman Perangkat Bergerak. Seluruh materi dan proyek di dalam repository ini dikembangkan menggunakan bahasa pemrograman Kotlin menggunakan Android Studio sebagai lingkungan pengembangannya.

Praktikum ini berfokus pada pengenalan konsep Kotlin lanjutan serta pengembangan antarmuka Android yang lebih kompleks. Materi diawali dengan pemahaman deklarasi variabel menggunakan kata kunci `val` dan `var`, dilanjutkan dengan penerapan fitur *Null Safety* untuk mencegah terjadinya *NullPointerException* (NPE) melalui penggunaan *Safe Call*, *Elvis Operator*, *Assertion*, dan *Safe Cast*. Selain itu, praktikum juga membahas struktur kontrol percabangan seperti `if`, `if-else`, `if-else if-else`, dan `when`, serta struktur kontrol perulangan menggunakan `for`, `while`, dan `do-while`.

Pada sisi pengembangan antarmuka (*User Interface*), repository ini memuat implementasi berbagai komponen Android seperti `LinearLayout` dengan orientasi vertikal dan horizontal, penggunaan atribut `match_parent` dan `wrap_content`, serta pemanfaatan widget interaktif seperti `RadioButton`, `Spinner` (Dropdown/Combo Box), `CheckBox`, `ImageView`, dan pengelolaan warna melalui berkas `colors.xml`. Materi juga memperkenalkan penggunaan aset gambar berupa *Image Asset* dan *Vector Asset* beserta karakteristik masing-masing.

Selain itu, praktikum ini mencakup implementasi *Intent* untuk perpindahan antar activity dan pertukaran data, serta penggunaan *AlertDialog* sebagai media interaksi antara aplikasi dan pengguna. Seluruh latihan dalam repository ini dirancang untuk meningkatkan pemahaman mahasiswa terhadap pengembangan aplikasi Android yang lebih dinamis, interaktif, dan sesuai dengan praktik pengembangan modern menggunakan Kotlin.
